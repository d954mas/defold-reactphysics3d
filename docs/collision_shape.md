# CollisionShape
This abstract class represents the collision shape associated with a body that is used during the narrow-phase collision detection.

You can create 6 types of shape:
1. CapsuleShape
2. SphereShape
3. BoxShape
4. ConvexMeshShape
5. ConcaveMeshShape
6. HeightFieldShape
